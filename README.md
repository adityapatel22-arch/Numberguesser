# Number Guesser Game 

A Python-based interactive number guessing game where the player attempts to guess a randomly generated number within a user-defined range. The game provides feedback after every guess and displays the total number of attempts at the end.

---

## Project Objective
The main goal of this project is to demonstrate the use of:
- User input handling
- Conditional logic
- Loops
- Random number generation
- Basic error handling in Python

This project was developed as part of the course requirement for learning fundamental Python programming concepts.
